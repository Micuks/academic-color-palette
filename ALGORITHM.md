# 学术配色推荐算法

## 概述

本系统使用 AI 模型生成每日配色方案，基于季节、节日、天气等多维度因素，创造出富有诗意和文化内涵的配色组合。

## 核心算法

### 1. 每日配色生成

**API 端点**: `/api/daily-palette`

**生成流程**:
1. 获取当前日期（Asia/Shanghai 时区）
2. 确定季节（春/夏/秋/冬）
3. 检查节日（传统节日、现代节日）
4. 调用 AI 模型生成配色方案

**AI Prompt 结构**:
```
你是一位专业的配色设计师。请为 {日期} 生成一个配色方案。

要求：
1. 配色名称：富有诗意，2-4 个汉字
2. 5 个颜色，每个颜色包含：
   - HEX 色值
   - 颜色名称（富有诗意）
   - RGB 值
3. 配色描述：100-150 字，描述配色意境
4. 使用场景：3-4 个应用场景

考虑因素：
- 季节：{季节}
- 节日：{节日列表}
- 星期：{星期几}
```

### 2. AI 配色生成

**模型**: 华为云 GLM-5 (huawei-v2/glm-5)

**生成参数**:
- temperature: 0.8
- max_tokens: 1000
- reasoning: true（显式思考模式）

**输出格式**:
```json
{
  "name": "破晓桃枝",
  "colors": [
    {"hex": "#F8F3E6", "name": "晨雾纱", "rgb": [248, 243, 230]},
    {"hex": "#E8B4B8", "name": "早樱颊", "rgb": [232, 180, 184]},
    {"hex": "#6A994E", "name": "新萝青", "rgb": [106, 153, 78]},
    {"hex": "#3C6E71", "name": "潭影碧", "rgb": [60, 110, 113]},
    {"hex": "#2F3E46", "name": "墨枝痕", "rgb": [47, 62, 70]}
  ],
  "description": "以清早桃林为意象，捕捉初春薄雾中花枝初绽、新绿暗涌的过渡瞬间...",
  "usage": ["🎨 平面设计", "📱 UI设计", "🏠 室内设计", "👗 时尚设计"],
  "festivals": []

```

### 3. 配色类型

系统支持以下配色类型：

1. **类似色** (Analogous)
   - 色轮上相邻的颜色
   - 和谐、统一的感觉

2. **互补色** (Complementary)
   - 色轮上相对的颜色
   - 强烈对比、视觉冲击

3. **三角配色** (Triadic)
   - 色轮上等距的三个颜色
   - 平衡、丰富

4. **四角配色** (Tetradic)
   - 色轮上等距的四个颜色
   - 复杂、多样

5. **自定义** (Custom)
   - 用户上传的配色方案
   - 社区共享

### 4. 颜色分析

**亮度调节**:
- 输入: HEX 颜色
- 输出: 不同亮度的颜色梯度
- 算法: HSL 色彩空间转换

**对比度计算**:
- WCAG 2.1 标准
- 对比度比率: 4.5:1（AA 级别）
