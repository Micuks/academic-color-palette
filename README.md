# 🎨 Academic Color Palette

现代美感的学术配色推荐器

## 设计理念
- **极简美学**: 减少文字，增加视觉元素
- **直接展示**: 无需繁琐设置，直接提供专业配色
- **实时交互**: 输入色码即可调整饱和度，实时预览
- **现代感**: 使用最新的设计趋势和动画效果

## 技术栈
- **前端框架**: React 18 + TypeScript
- **样式方案**: Tailwind CSS + Framer Motion
- **图标库**: Lucide React
- **颜色计算**: chroma-js
- **部署**: Vite + 静态托管

## 项目结构
```
academic-color-palette/
├── src/
│   ├── components/     # React组件
│   ├── pages/         # 页面组件
│   ├── utils/         # 工具函数
│   ├── styles/        # 样式文件
│   └── data/          # 配色数据
├── public/            # 静态资源
└── package.json       # 项目配置
```

## 核心功能
1. **配色库展示** - 按学科分类的专业配色
2. **饱和度调整器** - 实时调整颜色饱和度
3. **实时预览** - 即时查看调整效果
4. **导出功能** - CSS/LaTeX代码生成

## 设计原则
1. **Less Text, More Visual** - 减少文字，增加视觉元素
2. **Instant Feedback** - 所有操作都有即时反馈
3. **Smooth Animations** - 使用流畅的动画过渡
4. **Modern Aesthetics** - 符合2026年设计趋势
